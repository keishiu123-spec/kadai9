# ①課題名
地域安全プラットフォーム「まちの目」　完成版

## ②課題内容（どんな作品か）
＜WHO＞<br>
・毎朝「無事に帰ってきて」と祈りながら子供を送り出す保護者<br>
・通学路の安全対策をしたいが「根拠となるデータ」がなくて動けない自治体・学校関係者・警察署<br>
・地域の安全を守りたいと願うすべての市民<br>
＜WHAT＞<br>
・子供と共により安全な街に住むことができる<br>
＜HOW＞<br>
・警察の目が届かない場所でのスピード違反/一時不停止/歩行横断妨害等の違反を可視化し、データとして蓄積する。<br>
　1. ヒヤリハットのリアルタイム可視化（デジタル地図生成）と警戒資産の蓄積<br>
　2. 蓄積されたヒートマップを「要望書」として自動出力し、取り締まりの優先度を決定<br>
　3. 自動カメラの設置や交通違反予測の基盤へ<br>
＜COPY＞<br>
あなたの「気づき」が、街の防犯カメラになる。<br>
＜BODY＞<br>
街に潜む危険は、警察にも防犯カメラにもすべては見えていません。<br>
ですが、毎日そこを歩く親たちの目には、はっきりと映っています。<br>
「まちの目」は、あなたの小さな違和感をデータとして繋ぎ、抑止力に変えるプラットフォームです。 <br>
ヒヤリハットを可視化し、行政と連携することで、事故が起きる前に街を書き換える。<br>
テクノロジーで、あの子の通学路を世界で一番安全な場所にします。<br>
